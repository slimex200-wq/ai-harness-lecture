# AI 작업 하네스 강의자료

비개발자/기획자를 위한 AI 작업 하네스 강의 HTML 슬라이드입니다.

## 열기

브라우저에서 `index.html`을 열면 됩니다.

## GitHub Pages 배포

1. 이 폴더를 GitHub 저장소로 올립니다.
2. GitHub 저장소의 `Settings` -> `Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, folder는 `/root`로 설정합니다.
5. 생성된 GitHub Pages URL로 접속합니다.

필수 파일:

- `index.html`
- `agents_claude_design_reveal.html`
- `lecture_assets/`
